# VS Code (setup rápido)
Extensões recomendadas estão em `.vscode/extensions.json`.

## Passos
1- Abra a pasta do projeto no VS Code  
2- Crie/ative o venv  
3- Selecione o interpretador Python do `.venv`  
4- Rode:
```bash
make lint
make test
```
