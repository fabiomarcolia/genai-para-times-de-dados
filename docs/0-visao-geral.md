# Visão geral
Este repositório é um **playbook prático** para aplicar GenAI/LLMs em produtos e rotinas de dados, com foco em:

1- Valor de negócio (casos de uso reais)  
2- Engenharia (reprodutibilidade, testes, CI)  
3- Segurança (PII, governança, auditoria)  
4- Custo (offline-first, limites e monitoramento)

Se você está criando um portfólio, este repo foi pensado para “provar”:
- Você entende **arquitetura**
- Você entrega **projetos executáveis**
- Você explica **como ensinar e escalar**

Sugestão: use o README como landing page do seu portfólio.
